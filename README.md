# NDI Community Development Website

A professional organizational website for NDI Community Development Projects.

## Project Overview

The NDI Community Development website is a modern, responsive platform designed to showcase our organization's mission, projects, and community engagement initiatives.

## Features

✨ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
✨ **Modern Aesthetics** - Professional gradient design with smooth animations
✨ **Easy Navigation** - Intuitive menu structure for quick access to all sections
✨ **Project Showcase** - Display ongoing and completed community projects
✨ **Contact Integration** - Easy ways for community members to get involved
✨ **SEO Optimized** - Proper semantic HTML for better search engine visibility

## File Structure

```
NDI-website-/
├── index.html          # Main landing page
├── about.html          # Organization information
├── projects.html       # Project showcase
├── contact.html        # Contact and involvement page
├── styles.css          # Main stylesheet
└── README.md           # This file
```

## Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/twati807-ui/NDI-website-.git
   cd NDI-website-
   ```

2. **Open Locally**
   - Simply open `index.html` in your web browser
   - No build process or dependencies required

3. **View Online**
   - Visit the GitHub Pages deployment (if enabled)

## Customization

To customize the website:

1. **Update Content** - Edit HTML files directly with your content
2. **Change Colors** - Modify the gradient colors in `styles.css` (currently purple gradient: #667eea to #764ba2)
3. **Add Pages** - Create new .html files following the existing structure
4. **Update Navigation** - Modify nav links in all HTML files

## Color Scheme

- **Primary Gradient**: #667eea to #764ba2 (Purple)
- **Text Color**: #333 (Dark Gray)
- **Background**: #f9f9f9 (Light Gray)
- **Accent**: #667eea (Blue Purple)
